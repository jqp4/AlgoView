# AlgoView (2.0)

## Node JS Server

### 1. Install dependencies

```bash
npm install
```

### 2. Create folder uploadFiles

```bash
mkdir uploadFiles
```

### 3. Run web server in dev mode

```bash
npm run dev
```

### 4. Go to <http://localhost:3001>
